## 学习如何学习

主要原则：不能无恒（三天打鱼两天晒网）、不能贪多、不能求速成、不要怕吃苦（大腿臀部肌肉的酸疼）、不要怕吃亏（输了以后发急和人用蛮力）、不要怕看上去厉害的敌人

- [ ] 用身体和心灵学习，而不是用脑子推理，脑子推理分析是身心感受到以后的结果

#### 如何教别人和教他人太极拳

+ 手眼身法步顺序是要倒过来的，先练脚就像跳舞先练Steps。练脚的时候手背在后面或者抱在前面假装自己没手。再练身腰的旋转,engage 髂腰、腹外、腹内、腹横适当放松下背部。再练手（锁骨、肩胛、肱骨）。最后练眼，眼看中指。最后练用法。一层层往上加。

## 太极拳基本术语

+ 眼法：视野范围150度左右，除了有主意的点目标外，还要有面。顾处是实处，盼处是虚处。不能眼里只有目标，没有周围视野。
+ 掤: 一是指劲。不是力。二是指迎接对方来手的着法，以引化为主。***掤劲是太极拳的第一劲。掤劲是指用三角形成一种三角结构，三角能形成稳定与支撑。类似于跳舞中的Frame。腕、肘、肩三个点形成三角，三角结构产生一种劲。***能够保持掤劲不是指对方一碰你肘就弯曲，夹角变小，结构就不能维持。肘不能弯曲，缩进来。练的时候想像有人推你的小臂。肘关节不能主动弯曲。。肩胛骨活动，腰活动，但三角不能瘪掉。同样，脚腕、膝盖和胯部腰形成拱桥结构对自己的身体形成一种支撑，膝盖和胫骨不能软不能晃动。
+ Awareness和Concentration
  + Awareness能够注意力集中在一点，但是注意力的宽度增加了(Scope of attention widened)
  + Concentration能够注意力集中在一点，但是注意力的宽度缩小了(Scope of attention narrowed)

## 太极核心技术

### 一、把气沉入丹田(sink the chi to dantian)
- [ ] [Sink the chi to dantian](https://www.youtube.com/watch?v=C3Ucw2BfETQ)
  + 肩的内外旋不能连到小臂和脖子，通过后背和腹部练到胯。同理胯的内外旋不能连到小腿，通过腹部和腰连到肩
  + 气沉不下去，所有的动作都不对
  + 沉不下来，是肩这里被堵住，肩被堵住是因为脖子侧面后面前面都太紧，把肩拉得越来越高
- [ ] 每一动步沉着劲，不浮不起不晃
  + 分清楚支撑脚与移动脚，支撑脚要顺着重力踏实，移动脚要脚腕放松，膝盖不往上提，只是拉动小腿
- [ ] 让气流动
	+ 只有气能沉下来你才有东西用。左右前后压缩，上下只能伸展，脊柱延长，气到五点，头，指尖，脚尖。腹腔能够改变形状，改变气压，呼吸的能力就是改变腹腔形状的能力，但是不能用到脖子附近的肌肉去硬改变。
		- [ ] [Mobalize the chi](https://www.youtube.com/watch?v=1uK5uETFIy0)
		- [ ] [Mobalize the chi——容易犯的错误：](https://www.youtube.com/watch?v=fqr21SJlitg)仔细观察4:03女士的错误

### 二、太极就是阴阳
+ 时时刻刻分阴阳、分虚实、分动静
	- [ ] 哪个是支撑点，哪个是运动点，哪个是支点，哪个是动力源
+ 练太极就是“分”阴阳。就是上下、左右、前后、内外等的杠杆。
+ 就是分动静。身上一定同时有动的地方和不动的地方。就是将松散无序的身体变成有结构的、紧凑的如门轴一样的身体。
+ 要“分”就必须先“定”，定住一点，两头才能分开。这一点一开始在丹田，后来可以在身体的任何部分。这一点就是轴。可以是点，也可以是一条线。点是万向轴；线是单向轴。点外的也可是点（门把手）；或一个面（门面板）、一个体旋转门中间轴外都算）。

### 三、松胯
  + 胯是人体最大的关节，要是这个关节不松不灵活，所有关节都不灵活
  + 松胯是为了用胯，所有的动作都要Init from the kua，胯不松，没办法用到人体最具有力量的关节、筋和肌肉
  + 脚趾抓地，整个脚和腿有劲，胯才能松，裆才有劲

### 四、腿与根(Root)

- [ ] 脚不能睡着，脚要连到地，脚必须比手快
- [ ] [脚必须比手快](https://www.youtube.com/watch?v=1BsE8EvxgsI)
- [ ] 手必须跟随脚，否则就是上重下轻，是浮
- [ ] 凡推，劲通过腹部、后脚前侧与后脚掌连入大地。凡拉，劲通过背部（命门）、前腿后侧与前脚掌连入大地。
- [ ] 自己想要推时想象自己被推把劲接入大地，自己想要拉时想像自己被人从背后拉把劲接入大地。
### 五、梢节领劲，两头蜷曲

+ 手要做移动从A点到B点，大拇指或者小拇指带领，不是整个手动，其他四个手指要Reluctant，有点懒惰被带领。并且做动作前自己要有意识明确的意识你要去的B地是哪里

  参考视频：<https://www.youtube.com/watch?v=o5sCG2vSSX8&list=PLY8ILYkecJdvhaEkLZ7RaZ-rk8IBrKy01&index=1> 

+ 手指最后的一节要有略微弯曲之意，略微弯曲以后手可以启动下斜方肌，前锯肌。而不是手连倒颈椎。

### 六、坐腕、竖踝

+ 手腕与对手连接的点，手腕不松，就不能把能量传导到对方身上
+ 脚踝与大地连接的点，膝盖没有通过脚心把力传导到大地

### 七、松功三练习
+ 太极里所有的重心转换都是上下和旋转，没有左右前后移动
+ 手的左右前后运动与身体的上下不同步
+ 练围绕着百会穴一个点脊椎旋转，胯开始转，肩在胯里，膝盖放松，力量传到地底下

## 个人身心体悟

+ 观呼吸和坐姿、站姿、走姿、卧姿永远是最基本反复练习的基本功，永远与你同在

  + 提升动作前，先清楚了知我正在坐、我正在站、我正在蹲、我正在走
  + 回到爬行的婴儿状态，脚往后踢，一定有一股力量随着脊椎从尾骨一路冲到手到头
  + 脚的动作如果没有到手就是断劲了
  + 想像自己前面有个镜子像小孩一样趴着
+ 坐姿
  + 坐骨会贴着椅子像球一样滚动，带动骨盆，骨盆让腰椎有内凹的生理曲度
  + 坐骨的前面会有一个平面，坐在这个平面上
  + 坐骨的滚动不能靠腰部肌肉和颈部肌肉，要放松背部用髂腰肌
  + 不用颈部肌肉点头，用穿过耳朵和鼻子的想象线点头
  + 尽可能地对准骨盆和腰椎放松自己下背部地肌肉
+ 研究正视自己身上存在的问题才能摆脱问题。你自己的问题
  + 右侧骨盆往前侧旋
  + 骨盆后倾腰椎生理曲度不够，骨盆前倾和腰椎生理曲度是靠下肢做出来的
    + 体会站立时，下肢内旋鼓励骨盆前倾，下肢外旋骨盆后倾
    + 然而坐姿时，下肢外旋更鼓励骨盆前倾，下肢内旋会骨盆更容易后倾
  + 把骨架的各个位置摆正，通过旋转力让肌肉最少出力
+ 坐胯身体重量只压在口袋45度肌肉处，臀部和大腿后侧是松弛的就不对。一定是臀部后侧中侧前侧都用到。只有正确地坐胯了腰转才能达到手和脚。用Jeff的话Don't be lazy, sit into the kua。十年摸索出来，胯顶出来人就弱了就Weak没力量。(2020-06-13)
+ 脑海中永远永远要有一个清晰的意识，我要进入的下一个形状是什么。用什么元素进入下一个形状。不要太一点点控制中间的过程，反而会错。
+ **收肘绝不能把自己手与肩的联系在肘关节断了**，收肘一定要同时不收手。用前锯肌和肩关节下方的手臂在同一水平线上手肘。保持手到肩的联系不断。肩在往下接脚不是往下接颈椎。
+ **为什么我们看屏幕一会会儿眼睛就容易累？**因为我们总是用滚轮去滚屏幕，眼睛只是死死地盯住一点不动了。滚轮不动眼睛扫描。这样能舒缓颈椎的压力。
